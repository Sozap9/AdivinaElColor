# Adivina el color

## Introducción 

El proyecto "Adivivna el Color", es un proyecto del Bootcamp de Hack A Boss, esta basado en Javascript, CSS Y HTML. Su objetivo es seleccionar el color correcto que se corresponde con el RGB.
